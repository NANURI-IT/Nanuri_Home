# 나누리아이티 홈페이지 리팩토링 스펙 문서

> 기존 나누리아이티 홈페이지를 바탕으로, 금융 특화 정체성과 핵심 콘텐츠는 유지하면서,
> 메인 구조·문구·UI를 노션 스타일의 깔끔한 방향으로 정리하는 리팩토링 프로젝트다.

## 메인 페이지 구조

1. Hero Section - 회사 정체성 즉시 전달
2. Company Summary - 핵심 강점 요약
3. 사업영역 - 6개 카드형 구성
4. 핵심 솔루션 - IBIMS 등
5. 구축 사례 / 적용 분야
6. 신뢰 요소
7. 문의 CTA
8. Footer

## 사업영역 6개

- 금융 SI
- IB 시스템
- 고유자산
- 토큰증권
- 회계 서비스
- 채널 서비스

## 디자인 방향

- 노션 스타일 기반 깔끔하고 절제된 느낌
- 흰 배경 중심, 검정/짙은 회색 텍스트
- 포인트 컬러 1개
- 과한 모션/그래디언트/감성 디자인 금지
